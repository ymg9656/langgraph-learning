# Module Analyzer

학습 자료 .md 파일을 분석하여 실습 자료 제작에 필요한 구조화된 정보를 추출합니다.

## 핵심 기능

- 학습 목표(`- [ ]`) 자동 추출
- Python 코드 블록 추출 및 컨텍스트 라벨링
- 모듈 간 선수 지식 의존성 식별
- 실습 후보 코드 블록 자동 추천
- 난이도 평가 (S/M/L/XL)

## 워크플로우

### 1. 모듈 파일 읽기

대상 모듈 .md 파일을 전체 읽습니다.

```bash
# Read 도구로 파일 읽기
Read /Users/lee9656/workspace-github/langgraph-learning/docs/part{N}-{name}/{module}.md
```

### 2. 학습 목표 추출

`- [ ]` 패턴으로 학습 목표를 검색합니다.

```bash
# Grep으로 학습 목표 패턴 검색
Grep "- \[ \]" --path {module_file}
```

### 3. 코드 블록 추출

Python 코드 블록을 컨텍스트와 함께 추출합니다.

```bash
# 멀티라인 Grep으로 코드 블록 검색
Grep "```python" --path {module_file}
```

### 4. 분석 결과 구조화

추출된 데이터를 JSON 형태로 구조화합니다.

### 5. 실습 후보 추천

코드 블록 중 실습으로 변환 가능한 것을 식별하고 추천합니다.

## 에러 처리

| 에러 | 원인 | 해결 |
|------|------|------|
| 파일 없음 | 잘못된 모듈 경로 | docs/ 디렉토리 구조 확인 |
| 코드 블록 0개 | 개념 전용 모듈 (예: Module 01) | 퀴즈 형식으로 전환 |
| 학습 목표 0개 | 문서 형식 불일치 | 수동으로 섹션 헤더에서 목표 유추 |

## 사용 예시

### 예시 1: Module 02 분석

```
Module 02 (LangGraph 기초) 학습 자료를 분석하여
코드 블록, 학습 목표, 실습 후보를 추출해주세요.
```

### 예시 2: 전체 모듈 일괄 분석

```
docs/ 디렉토리의 모든 모듈을 분석하고
모듈 간 의존성 그래프를 생성해주세요.
```
