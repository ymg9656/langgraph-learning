# Coverage Checker

실습 자료가 원본 학습 자료의 모든 학습 목표를 빠짐없이 커버하는지 검증합니다.

## 핵심 기능

- 학습 목표(`- [ ]`) 자동 추출 및 추적
- 실습 노트북의 TODO/실습 항목과 학습 목표 매핑
- 정량적 커버리지 수치(%) 산출
- 미커버 항목 식별 및 리포트
- 코드 정확성 교차 대조

## 워크플로우

### 1. 원본 학습 목표 추출

학습 자료에서 모든 학습 목표를 추출합니다.

```bash
# 모든 모듈의 학습 목표 검색
Grep "- \[ \]" --path /Users/lee9656/workspace-github/langgraph-learning/docs/
```

### 2. 실습 노트북 항목 추출

실습 노트북에서 TODO 항목과 실습 제목을 추출합니다.

```bash
# 실습 노트북에서 TODO 검색
Grep "TODO" --path /Users/lee9656/workspace-github/langgraph-learning/practice/
```

### 3. 매핑 테이블 생성

학습 목표와 실습 항목을 1:1 매핑합니다.

### 4. 커버리지 계산

매핑 결과를 바탕으로 커버리지를 산출합니다.

### 5. 리포트 생성

검증 결과를 구조화된 리포트로 출력합니다.

## 에러 처리

| 에러 | 원인 | 해결 |
|------|------|------|
| 커버리지 0% | 실습 파일 미생성 | 실습 노트북 경로 확인 |
| 학습 목표 0개 | 모듈 경로 오류 | docs/ 디렉토리 구조 확인 |
| 매핑 불일치 | 학습 목표와 실습 주제 불일치 | 수동 매핑 검토 필요 |

## 사용 예시

### 예시 1: 단일 모듈 커버리지 체크

```
Module 02의 학습 목표 커버리지를 확인해주세요.
```

### 예시 2: 전체 프로젝트 커버리지 리포트

```
모든 모듈의 커버리지를 확인하고 전체 리포트를 생성해주세요.
```
